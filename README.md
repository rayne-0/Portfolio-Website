# Premium Developer Portfolio

A highly polished, interactive, and responsive developer portfolio designed with Next-Gen aesthetics. This project was built utilizing cutting edge modern web design principles to offer an engaging experience filled with micro-animations, glassmorphism, and dynamic lighting.

## 🚀 Features

*   **Deep Dark Theme**: Immersive `Slate & Indigo` color palette by default.
*   **Dynamic Glassmorphism**: Cards and navigation bars feature frosted-glass backdrop filtering for depth and elevation.
*   **Organic UI Elements**: 
    *   Floating plasma orb background effects.
    *   Responsive, glowing mouse-cursor tracker (blob).
*   **Intersection Animations**: Elements smoothly slide into view as you scroll down the page via JavaScript Intersection Observers.
*   **Infinite Tech Marquee**: A seamless, looping technology banner emphasizing a modern tech stack.
*   **Mock Admin Command Center**: (Client-side demo) Integrated secret modal to manage portfolio entries. Accessible via `Shift + A`.

## 🛠️ Technology Stack

Built completely with **Vanilla Web Technologies** (No complex frameworks or build steps required):

*   **HTML5**: Semantic, accessible document structure.
*   **CSS3**: Custom properties (variables), Flexbox/Grid layouts, advanced animations (`@keyframes`), and backdrop filters. 
*   **JavaScript (ES6+)**: DOM manipulation, event listeners, Intersection Observers, and LocalStorage for state management.
*   **Fonts**: Inter (Body) & Outfit (Headings) via Google Fonts.

## 📦 Setup & Usage

To view this portfolio, absolutely no build steps are necessary.

1.  **Clone or Download** the repository.
2.  **Open `index.html`** in any modern web browser.
3.  *(Optional)* For the best experience, use a local server (like VS Code's "Live Server" extension) to bypass strict cross-origin restrictions on local files (though it works perfectly fine by double-clicking the HTML file).

## 🔒 Secret Admin Portal Demo

To view the mock project management interface:

1.  Press `Shift + A` on your keyboard anywhere on the page to open the login modal.
2.  Enter the override code: `admin`
3.  Access the Command Center to visually add, edit, or delete the project entities on the display grid. 
> *Note: This management system relies purely on `localStorage` for demo purposes and is inherently insecure for actual production backends.*

## 🎨 File Architecture

*   `index.html`: The core architectural skeleton.
*   `styles.css`: The entire design system, variables, and animation logic.
*   `scripts.js`: The interactivity engine, intersection observers, and admin portal logic.
